# Django-test

Simple Cart Using Django Framework

## Getting Started

Mini Project using:
- Django 1.11
- Bootstrap 4

### Prerequisites

[Docker](https://docs.docker.com/install/), [Docker Compose](https://docs.docker.com/compose/install/)

### Running

```
docker-compose build
```

```
docker-compose up
```
